# Projeto de Algoritmos de Escalonamento de Processos

Projeto de Escalonamento – Sistemas Operacionais

Repositório criado para o projeto da disciplina de Sistemas Operacionais (2º ADS – IF Baiano). O objetivo é implementar quatro algoritmos de escalonamento em linguagem C, seguindo a estrutura fornecida pelo professor:

FCFS

SJF

Priority Scheduling

Round Robin

Cada aluno ficou responsável por um dos algoritmos, e as implementações estão na pasta /src. O projeto segue as orientações do repositório-base e será utilizado para a entrega do documento, vídeo de explicação e apresentação final.
